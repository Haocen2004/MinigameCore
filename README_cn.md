# MinigameCore

[English](./README.md) | 简体中文

Pluscraft 小游戏插件  
灵感来自 Dream 的视频

## 包含的游戏
翻译暂定，欢迎提出意见
* 刺杀者
* 方块占领
* 随机切换

## 指令
/minigame reload  
/assassin start/stop  
/randomray start/stop  
/shuffle start/stop  

## 运行环境
Spigot 1.15.2  
支持低版本 `需要修改 blocks.yml 删除高版本方块`

## 警告
我使用了自定义目录来方便我进行调试，所以默认编译路径是 `.\.debug\plugins`  
插件使用了spigot特性所以不支持纯bukkit服务器
